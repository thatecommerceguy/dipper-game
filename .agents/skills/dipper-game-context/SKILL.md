# Dipper Game Context

Ground work in the supplied reference material without treating every sketch as a finalized requirement.

## Load the project brief

Read [references/game-context.md](references/game-context.md) before doing substantial work on the game.

Inspect images in `../../../reference/` when the task depends on visual details. Use the reference index in the project brief to select relevant images instead of loading every image for every task.

## Apply the context

1. Preserve explicit user decisions over all notes and interpretations.
2. Treat the project as a 2D side-scrolling platform game with the round Dipper mascot.
3. Use Past, Present, and Future as the established world direction, while keeping their progression and final implementation open until confirmed.
4. Keep visual work consistent with the era palettes and materials described in the brief.
5. Keep gameplay proposals compatible with the referenced movement, platforms, hazards, collectibles, powers, timer, audio, and safe-zone ideas.
6. Label design assumptions when a task depends on an unresolved detail.

## Maintain decision integrity

- Do not promote a sketch, generated image, question, or crossed-out note into a confirmed requirement.
- Do not assume that monetization, signup, payment, leaderboard, or reward ideas are approved.
- Do not choose the final controls, attack behavior, damage penalty, win condition, screen format, or era progression without user direction when that choice materially affects the result.
- Treat later explicit user instructions and accepted implementation decisions as newer sources of truth.
- Update the project brief when the user resolves an open question or materially changes the concept.

## Use the visual references

- Use character sheets to understand motion and personality, not as production-ready sprite sheets unless the user approves them.
- Use photographs and screenshots as mood, palette, texture, and material references. Do not copy embedded UI elements from screenshot tools.
- Preserve the strongest era contrast:
  - Past: ornate gold, burgundy, baroque luxury.
  - Present: white forms, blue linework, sky, clouds, notebook/blueprint character.
  - Future: smooth white forms and realistic clouds; details remain comparatively undefined.
